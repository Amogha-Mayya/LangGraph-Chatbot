# LangGraph AI Chatbot

AI chatbot built with LangGraph featuring real-time response streaming, persistent memory, tool calling, MCP integration, and SQLite-backed state management for seamless conversational experiences.

## Features

* Real-time Response Streaming
* Short-Term & Long-Term Memory
* Persistent Conversation History
* Chat Resumption
* SQLite State Management
* Tool Calling
* Model Context Protocol (MCP)
* Stateful Agent Workflows
* Streamlit Interface

## Technologies Used

* Python
* LangGraph
* LangChain
* Streamlit
* SQLite
* MCP (Model Context Protocol)
* OpenAI / LLM APIs
